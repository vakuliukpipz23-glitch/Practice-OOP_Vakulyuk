# Practice-OOP_RPG

Компактний консольний RPG-симулятор для навчальної практики з об’єктно-орієнтованого програмування.

## Структура проєкту

- `Domain/PracticeOOP_RPG.Domain` — доменні моделі, ієрархія персонажів, зброя та бойова логіка.
- `App/PracticeOOP_RPG.App` — консольна програма для демонстрації бою.
- `Tests/PracticeOOP_RPG.Tests` — юніт-тести для перевірки логіки бою та поведінки патернів.

## Запуск

1. Відкрити кореневу папку `Practice-OOP_Vakulyuk`.
2. Виконати `dotnet run --project App/PracticeOOP_RPG.App/PracticeOOP_RPG.App.csproj`.
3. Виконати тести: `dotnet test`.

## Особливості

- Впроваджено `Strategy` для різних типів атак (melee та spell).
- Впроваджено `Factory` для створення типових ворогів та зброї.
- Використано події для логування бою (`Observer`-подібна модель).
