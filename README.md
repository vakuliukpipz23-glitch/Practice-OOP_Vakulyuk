# Practice-OOP_RPG

Цей проєкт реалізує просту RPG-систему з об'єктно-орієнтованою архітектурою на C#.

## Що зроблено

- Створено доменну модель персонажів: `Character`, `PlayerCharacter`, `Monster`.
- Реалізовано інтерфейс `ICharacter` для уніфікованої роботи з персонажами.
- Додано озброєння: `Weapon` та логіку його застосування для розрахунку атаки.
- Реалізовано можливості персонажів у вигляді `Ability` з підтримкою витрати мани та стратегії атаки.
- Впроваджено патерн "Стратегія" для атак: `IAttackStrategy`, `MeleeAttackStrategy`, `SpellAttackStrategy`, `HeavyStrikeStrategy`.
- Додано систему бою: `Battle`, яка проводить бій між двома персонажами.
- Побудовано допоміжні класи для керування грою: `Party`, `Inventory`, `Item`.
- Реалізовано серіалізацію стану гри через `GameStateDto`, `GameStateSerializer` та `GameStateXmlSerializer`.
- Створено фабрики для створення персонажів, ворогів, зброї та здібностей: `CharacterFactory`, `EnemyFactory`, `WeaponFactory`, `AbilityFactory`.

## Структура проекту

- `App/PracticeOOP_RPG.App/` — консольний додаток, який використовує доменну логіку.
- `Domain/PracticeOOP_RPG.Domain/` — бізнес-логіка гри та модель.
- `Tests/PracticeOOP_RPG.Tests/` — модульні тести для перевірки функціоналу.

## Як використовувати

1. Відкрити рішення `PracticeOOP_RPG.sln`.
2. Запустити проект `PracticeOOP_RPG.App` у Visual Studio.
3. За необхідності додати або змінити логіку у доменному проекті.
